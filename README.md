# Snake

![](images/preview.png)

## Description
This is the simple implementation of a snake game using C and SDL2 as a drawing engine. Tested only on Linux systems

## Requirements

- SDL2
- CMake


## Installation

Simply run these 3 commands
```
cmake .
make
./CSnake
```

## Controls

- Arrows - Movement
- Spacebar - Pause / Resume
- Esc - Exit
